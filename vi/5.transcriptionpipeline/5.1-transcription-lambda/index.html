Transcription Lambda

Phần này, chúng ta sẽ tạo một AWS Lambda function để tạo phụ đề cho video. Lambda sẽ nhận thông báo từ Transcription SQS và gọi Transcribe API để tạo phụ đề cho video.

Các bước gần như y hệt như khi tạo Lambda cho Transcoding, chỉ khác ở code và cấu hình IAM Roles.

Tạo Lambda Function

Truy cập AWS Lambda Console và chọn Create function.

Điền các thông tin như sau:

Chọn Create function để tạo Lambda.

Thêm Trigger cho Lambda

Ta sẽ biến Transcoding SQS thành trigger cho Lambda.

Đầu tiên phải cấp quyền cho Lambda để nhận message từ SQS. Truy cập Lambda vừa tạo và vào tab Configuration rồi chọn vào Role của Lambda.

Ở giao diện IAM Roles của Lambda, vào tab Permissions và chọn Attach policies.

Tìm kiếm SQS và chọn AWSLambdaSQSQueueExecutionRole. Role này cho phép Lambda nhận message từ SQS.

Chọn Add permissions để thêm policy.

Ngoài ra, cần thêm policy cho Lambda để truy cập Transcribe API. Tìm kiếm Transcribe và chọn AmazonTranscribeFullAccess. Chọn Add permissions để thêm policy.

Quay lại tab Configuration của Lambda và chọn Add trigger.

Chọn SQS, chọn Transcription SQS đã tạo và điền thông tin như sau:

Sau đó chọn Add để thêm trigger.

Code Lambda Function

Chọn tab Code và điền code như sau:

import boto3
import os
import json
import random

def parse_messages_and_transcribe(event, context):
    # Initialize Elastic Transcoder client
    transcribe = boto3.client('transcribe')

    # Get the S3 bucket and object key from the event
    for message_record in event['Records']:
        # Parse the body from JSON string to a dictionary
        body = json.loads(message_record['body'])

        # Access the SNS message part of the body
        sns_message = json.loads(body['Message'])

        for record in sns_message['Records']:
            s3_bucket = record['s3']['bucket']['name']
            s3_key = record['s3']['object']['key']
            s3_uri = f"s3://{s3_bucket}/{s3_key}"

            # Define output file paths
            base_filename = s3_key.rsplit('.', 1)[0]

            transcribe_job_name = f"transcription-{base_filename}-{random.randint(0,10000)}"
            transcribe.start_transcription_job(
                TranscriptionJobName=transcribe_job_name,
                Media={'MediaFileUri': s3_uri},
                MediaFormat='mp4',
                OutputBucketName = os.environ['S3_OUTPUT'],
                OutputKey = f"{base_filename}/transcription/{base_filename}",
                LanguageCode = 'en-US',
                Subtitles = {
                    'Formats': [
                        'vtt','srt'
                    ],
                    'OutputStartIndex': 1
               }
            )

        print(f"Transcription job {transcribe_job_name} started")

def lambda_handler(event, context):
    try:
        parse_messages_and_transcribe(event, context)
    except Exception as ex:
        print(ex)
        return 1

Nhiệm vụ của code trên là nhận message từ SQS, lấy thông tin về video từ message và gọi Transcribe API để tạo phụ đề cho video. Để biết thêm về các tham số của Transcribe API, xem tài liệu ở đây.

Sau đó chọn Deploy để lưu code.

Environment Variables

Trong đoạn code trên, chúng ta sử dụng biến môi trường S3_OUTPUT để lưu thông tin về bucket lưu phụ đề. Để thêm biến môi trường, chọn tab Configuration và chọn Environment variables.

Chọn Edit và thêm biến môi trường S3_OUTPUT với giá trị là tên của Output Bucket.
